What is Bootstrap?

Bootstrap is a powerful front-end framework for faster and easier web development. It includes HTML, CSS, and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Containers

Bootstrap provides two types of containers:

  • .container: A responsive fixed-width container.
  • .container-fluid: A full-width container spanning the entire width of the viewport.
